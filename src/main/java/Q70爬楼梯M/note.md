尝试寻找问题中的重复性:

| 层数 | 爬法            |                                                              |
| ---- | --------------- | ------------------------------------------------------------ |
| 1    | 1               | 走1步                                                        |
| 2    | 2               | 1+1或2                                                       |
| 3    | f(1) + f(2)     | 到第3层一定是：①从第1层走2步到 或者 ②从第2层走1步到。分开考虑这两种情况，可以得到f(3) = f(1) + f(2) |
| 4    | f(2) + f(3)     |                                                              |
| ...  | ...             |                                                              |
| n    | f(n-1) + f(n-2) |                                                              |
